About

This project is a Kanban-style project management tool designed to help users organise tasks and track progress efficiently.

Users can create projects, break them into tasks, and manage their workflow using a simple and intuitive drag-and-drop interface.

The goal of this system is to provide a clean, lightweight solution for managing personal and team projects without unnecessary complexity.
